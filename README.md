## ‚ôªÔ∏è About The Project

**JRS Recycling Solutions** is a dedicated platform for managing electronic waste. This application serves as the digital front for the business, providing information about their services, compliance standards, and environmental commitment.

The website is designed to be fast, responsive, and user-friendly, ensuring that potential clients can easily navigate through services like Data Destruction, IT Asset Disposition, and Recycling processes.

## ‚ú® Key Features

*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
*   **Modern UI/UX**: Clean interface with smooth animations using Framer Motion.
*   **Service Showcase**: Detailed sections for various recycling and disposal services.
*   **Compliance & Security**: Dedicated pages explaining data security protocols and certifications.
*   **Interactive Elements**: Animated statistics, testimonial carousels, and contact forms.
*   **Direct Communication**: Integrated WhatsApp button for instant customer support.

## üöÄ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

*   **Node.js**: Ensure you have Node.js installed (v16 or higher recommended).
*   **npm**: npm is usually installed with Node.js.

### Installation

1.  **Clone the repository**
    ```sh
    git clone https://github.com/your-username/jrs-recycling-solutions.git
    ```

2.  **Navigate to the project directory**
    ```sh
    cd jrs-recycling-solutions
    ```

3.  **Install dependencies**
    ```sh
    npm install
    ```

4.  **Start the development server**
    ```sh
    npm run dev
    ```

5.  **Open your browser**
    Navigate to `http://localhost:5173` (or the port shown in your terminal).
